# smartnotez
Erstelle mir eine Webapp auf Basis von pyhton und flask die in Anlehnung an Obsidian es mir ermöglicht Notitzen zu erstellen und zu verwalten. 
- Die Notizen sollen als .md also Markdown Dateien gespeichert werden.
- Jede Notiz soll mit einem oder mehreren Label versehen werden können. 
- Für neue oder bestehende Notizen soll es eine Spracherkennungsfunktion geben. Die Spracherkennungsfunktion soll über einen Button mit Mikrophon Symbol aufgerufen werden können. Die Spracherkennung soll mit einer über .env Funktion konfigurierbaren KI (z.B. Openrouter) funktionieren. Für neue Notizen soll automatisch eine Bewertung der KI stattfinden, die die Notiz mit den passenden, aber nur existierenden, Labels versieht.
